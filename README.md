# BMI Calculator

A modern, responsive **Body Mass Index (BMI) Calculator** built using **HTML, CSS, and  JavaScript**.  
The app allows users to input their height and weight, instantly calculates BMI, and displays a clear health category with helpful guidance.

---

##  Features

-  Height input in centimeters
-  Weight input in kilograms
-  Accurate BMI calculation using the standard formula
-  Health classification:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
-  Clean, modern UI with glassmorphism styling
-  Instant results with no page reload
-  Fully client-side 

---

##  Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling, layout, animations
- **JavaScript** – BMI calculation and DOM manipulation
